# Vibe Coding CheatSheet

> 面向新手的 AI 编程系统性入门 · Cyberpunk × Terminal × Pocket Dictionary

一份给 Vibe-coding 新手的高密度速查页面。中英双语、全键盘操作、可一键复制提示词模板。

## Quick start

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # 产出 dist/
npm run preview  # 预览生产构建
```

## Stack

- **Vite + React + TypeScript** — 单页应用
- **Tailwind CSS** — 主题/原子化样式
- **Fuse.js** — 全局模糊搜索
- **JetBrains Mono / Orbitron** — 终端等宽 + 赛博朋克标题字体

## 交互

- `Ctrl+K` / `/` — 全局搜索（标题、术语、模板、模型、工具）
- 代码 / 提示词块 hover 显示 `[ COPY ]`
- 顶部 `中 / EN` 切换语言（持久化）
